<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">Quick Start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">Get your asset management system up and running in minutes</p>
  
  <Steps>
    <Step title="Set up the database">
      Configure your MySQL database and run the initialization scripts to create the required tables for assets, suppliers, purchase orders, and SUNARP registrations.
      
      ```sql
      CREATE DATABASE bienes_db;
      USE bienes_db;
      ```
    </Step>
    
    <Step title="Configure the connection">
      Update the database connection settings in your deployment configuration to connect to your MySQL instance.
      
      ```java
      // MySqlConexion.java
      public static Connection getConectar() {
        String url = "jdbc:mysql://localhost:3306/bienes_db";
        return DriverManager.getConnection(url, user, password);
      }
      ```
    </Step>
    
    <Step title="Deploy the application">
      Deploy the web application to your Java servlet container (Tomcat, JBoss, etc.) and access the login page to start managing your assets.
    </Step>
    
    <Step title="Start managing assets">
      Log in with your operator credentials and begin tracking assets, managing suppliers, and processing purchase orders through the web interface.
    </Step>
  </Steps>
</div>
