## Overview

The `Proveedor` entity represents suppliers who provide goods and services to the organization. It stores supplier identification, contact information, location, and status.

**Package:** `com.bienes.entidad`

**Imports:** None

## Usage Example

### Creating a Proveedor Instance

```java
import com.bienes.entidad.Proveedor;

// Create a new supplier
Proveedor proveedor = new Proveedor();
proveedor.setNom_prove("Tech Supplies S.A.");
proveedor.setDist_prove("San Isidro");
proveedor.setTelf_prove("01-2345678");
proveedor.setEstado_prove("Activo");
```

### Using with DAO Layer

```java
import com.bienes.interfaces.ProveedorDAO;
import com.bienes.dao.MySqlProveedorDAO;

// Initialize DAO
ProveedorDAO dao = new MySqlProveedorDAO();

// Save a new supplier
int result = dao.save(proveedor);

// Find supplier by ID
Proveedor foundProveedor = dao.findById(1);

// Update supplier information
foundProveedor.setTelf_prove("01-9876543");
dao.update(foundProveedor);

// List all suppliers
List<Proveedor> allProveedores = dao.listAll();

// Delete a supplier
dao.delete(1);
```
