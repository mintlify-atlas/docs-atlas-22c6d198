## Overview

The `Menu` entity represents navigation menu items in the asset management system. It defines the menu structure, URLs, and icons for the sidebar navigation based on user permissions.

**Package:** `com.bienes.entidad`

## Fields Reference

<ParamField path="codigo" type="int">
  Unique identifier for the menu item (primary key)
</ParamField>

<ParamField path="nombre" type="String">
  Display name of the menu item (e.g., "Gestión de Bienes", "Proveedores")
</ParamField>

<ParamField path="url" type="String">
  Navigation URL for the menu item (e.g., "Bienes.jsp", "proveedor.jsp")
</ParamField>

<ParamField path="icon" type="String">
  Icon class for the menu item (Boxicons class, e.g., "bx bx-box")
</ParamField>

## Usage Example

### Loading User Menu

```java
// Retrieve menu items for logged-in user
LoginServices service = new LoginServices();
List<Menu> menuItems = service.obtenerMenu(userId);

// Store in session for sidebar rendering
session.setAttribute("LISTA", menuItems);
```

### Menu Rendering (JSP)

```jsp
<!-- Display menu in sidebar navigation -->
<c:forEach items="${sessionScope.LISTA}" var="bean">
    <li class="nav-link">
        <a href="${bean.url}">
            <i class="${bean.icon} icon"></i>
            <span class="text nav-text">${bean.nombre}</span>
        </a>
    </li>
</c:forEach>
```

## Database Table

The `Menu` entity maps to the `menu` table:

| Column | Type | Description |
|--------|------|-------------|
| `codigo` | INT | Menu item ID (primary key) |
| `nombre` | VARCHAR | Menu item display name |
| `url` | VARCHAR | Target URL/page |
| `icon` | VARCHAR | Boxicons CSS class |
