## Overview

The `Bienes` entity represents goods or assets that are tracked in the system. It contains information about the item description, quantity, supplier, arrival date, and its associated purchase order.

**Package:** `com.bienes.entidad`

**Imports:**
- `java.sql.Date`

## Usage Example

### Creating a Bienes Instance

```java
import com.bienes.entidad.Bienes;
import java.sql.Date;

// Create a new Bienes instance
Bienes bien = new Bienes();
bien.setDescrip_bien("Laptop Dell Latitude 5420");
bien.setCantidad_bien(10);
bien.setNom_provee("Tech Supplies S.A.");
bien.setFecha_llegada(Date.valueOf("2026-03-15"));
bien.setCodigoOrdendeCompra(12345);
```

### Using with DAO Layer

```java
import com.bienes.interfaces.BienesDAO;
import com.bienes.dao.MySqlBienesDAO;

// Save a new asset
BienesDAO dao = new MySqlBienesDAO();
int result = dao.save(bien);

// Find by ID
Bienes foundBien = dao.findById(1);

// List all assets
List<Bienes> allBienes = dao.listAll();

// List assets by purchase order
List<Bienes> bienesPorCompra = dao.listarBienesporCodigodeComptra(12345);
```
