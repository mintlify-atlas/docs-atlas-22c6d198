## Overview

The `InscripcionSUNARP` entity represents registrations with SUNARP (Superintendencia Nacional de los Registros Públicos), the Peruvian public registry system. It tracks asset information that needs to be registered including descriptions, quantities, prices, suppliers, and arrival dates.

**Package:** `com.bienes.entidad`

**Imports:**
- `java.sql.Date`

## Usage Example

### Creating an InscripcionSUNARP Instance

```java
import com.bienes.entidad.InscripcionSUNARP;
import java.sql.Date;

// Create a new SUNARP registration
InscripcionSUNARP inscripcion = new InscripcionSUNARP();
inscripcion.setDescrip_bien("Vehículo Toyota Hilux 2025");
inscripcion.setCantidad_bien(1);
inscripcion.setPrecio_bien(125000.50);
inscripcion.setNom_provee("Automotores del Peru S.A.");
inscripcion.setFecha_llegada(Date.valueOf("2026-02-20"));
```

### Using with DAO Layer

```java
import com.bienes.interfaces.InscripcionSUNARPDAO;
import com.bienes.dao.MySqlInscripcionSUNARPDAO;

// Initialize DAO
InscripcionSUNARPDAO dao = new MySqlInscripcionSUNARPDAO();

// Save a new SUNARP registration
int result = dao.save(inscripcion);

// Find by ID
InscripcionSUNARP found = dao.findById(1);

// Update registration
found.setPrecio_bien(128000.00);
dao.update(found);

// List all registrations
List<InscripcionSUNARP> allInscripciones = dao.listAll();

// Delete a registration
dao.delete(1);
```

## About SUNARP

<Info>
  SUNARP (Superintendencia Nacional de los Registros Públicos) is Peru's National Superintendency of Public Registries. This entity handles the registration of assets that require public registration such as vehicles, real estate, and other significant property.
</Info>
