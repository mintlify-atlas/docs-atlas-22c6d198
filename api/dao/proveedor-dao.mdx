## Overview

The `ProveedorDAO` interface and its MySQL implementation provide CRUD operations for managing **proveedores** (suppliers) in the system. Suppliers are essential entities linked to assets, purchase orders, and other business operations.

### Interface

**Package**: `com.bienes.interfaces`

```java ProveedorDAO.java
public interface ProveedorDAO {
    public int savePro(Proveedor bean);
    public int updatePro(Proveedor bean);
    public int deletePro(int id);
    public Proveedor findByIdPro(int id);
    public List<Proveedor> listAll();
}
```

### Implementation

**Package**: `com.bienes.dao`  
**Class**: `MySqlProveedorDAO`  
**Location**: `com/bienes/dao/MySqlProveedorDAO.java`

## Database Schema

### Table: `proveedor`

| Column | Type | Description |
|--------|------|-------------|
| `id_prove` | INT (PK, AUTO_INCREMENT) | Unique supplier identifier |
| `nom_prove` | VARCHAR | Supplier name/company |
| `dist_prove` | VARCHAR | District/location |
| `telf_prove` | VARCHAR | Phone number |
| `estado_prove` | VARCHAR | Status (active/inactive) |

## Usage Examples

### Creating a New Supplier

```java
// Obtain DAO instance
DAOFactory factory = DAOFactory.getDAOFactory(DAOFactory.MYSQL);
ProveedorDAO dao = factory.getProveedorDAO();

// Create supplier object
Proveedor proveedor = new Proveedor();
proveedor.setNom_prove("TechSupply S.A.");
proveedor.setDist_prove("Lima");
proveedor.setTelf_prove("987654321");
proveedor.setEstado_prove("Activo");

// Save to database
int result = dao.savePro(proveedor);
if(result > 0) {
    System.out.println("Supplier created successfully");
}
```

### Listing Active Suppliers

```java
ProveedorDAO dao = factory.getProveedorDAO();
List<Proveedor> proveedores = dao.listAll();

for(Proveedor p : proveedores) {
    if("Activo".equals(p.getEstado_prove())) {
        System.out.println(p.getNom_prove() + " - " + p.getTelf_prove());
    }
}
```

### Soft Delete (Deactivate Supplier)

```java
ProveedorDAO dao = factory.getProveedorDAO();
Proveedor proveedor = dao.findByIdPro(15);

if(proveedor != null) {
    proveedor.setEstado_prove("Inactivo");
    int result = dao.updatePro(proveedor);
    if(result > 0) {
        System.out.println("Supplier deactivated successfully");
    }
}
```

## Status Values

Common status values used in `estado_prove`:

| Status | Description |
|--------|-------------|
| `Activo` | Active supplier, available for orders |
| `Inactivo` | Inactive/deactivated supplier |
| `Suspendido` | Temporarily suspended |
| `Bloqueado` | Blocked due to issues |

<Note>
Status values are stored as strings. Consider using an enum or lookup table for better data integrity.
</Note>
