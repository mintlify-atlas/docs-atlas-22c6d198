## Overview

The `InscripcionSUNARPDAO` interface and its MySQL implementation provide CRUD operations for managing **inscripciones SUNARP** (Public Registry inscriptions). SUNARP (Superintendencia Nacional de los Registros Públicos) is Peru's national public registry system for property and asset registration.

### Interface

**Package**: `com.bienes.interfaces`


## Usage Examples

### Creating a SUNARP Inscription

```java
// Obtain DAO instance
DAOFactory factory = DAOFactory.getDAOFactory(DAOFactory.MYSQL);
InscripcionSUNARPDAO dao = factory.getInscripcionSUNARPDAO();

// Create inscription object
InscripcionSUNARP inscripcion = new InscripcionSUNARP();
inscripcion.setDescrip_bien("Terreno Urbano - Miraflores");
inscripcion.setCantidad_bien(1);
inscripcion.setPrecio_bien(500000.00);
inscripcion.setNom_provee("Inmobiliaria Lima S.A.");
inscripcion.setFecha_llegada(new Date(System.currentTimeMillis()));

// Save to database
int result = dao.save(inscripcion);
if(result > 0) {
    System.out.println("SUNARP inscription created successfully");
}
```

### Retrieving All Inscriptions

```java
InscripcionSUNARPDAO dao = factory.getInscripcionSUNARPDAO();
List<InscripcionSUNARP> inscripciones = dao.listAll();

double totalValue = 0;
for(InscripcionSUNARP ins : inscripciones) {
    System.out.println(ins.getDescrip_bien() + " - $" + ins.getPrecio_bien());
    totalValue += ins.getPrecio_bien();
}
System.out.println("Total registered value: $" + totalValue);
```

### Updating Inscription Value

```java
InscripcionSUNARPDAO dao = factory.getInscripcionSUNARPDAO();
InscripcionSUNARP inscripcion = dao.findById(42);

if(inscripcion != null) {
    // Update property value
    inscripcion.setPrecio_bien(550000.00);
    int result = dao.update(inscripcion);
    if(result > 0) {
        System.out.println("Inscription value updated");
    }
}
```

## Business Context

### SUNARP Overview

**SUNARP** (Superintendencia Nacional de los Registros Públicos) is Peru's national public registry system that manages:

- Property registrations
- Vehicle registrations
- Corporate records
- Asset ownership records
- Legal encumbrances

<Note>
This DAO manages inscription records within the application. Actual SUNARP registration requires official government processes.
</Note>

## Data Considerations

### Price Precision

The `precio_bien` field uses `DOUBLE` type. For financial applications, consider:

```java
// Use BigDecimal for precise financial calculations
BigDecimal precio = new BigDecimal(inscripcion.getPrecio_bien());
```

### Date Handling

The `fecha_llegada` field represents either arrival date or registration date depending on business logic. Ensure consistent interpretation across the application.
