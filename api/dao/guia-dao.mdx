## Overview

The `GuiaDAO` interface and its MySQL implementation provide CRUD operations for managing **guías y órdenes de compra** (purchase guides and orders). These records track purchases made from suppliers, including descriptions, quantities, and pricing.

### Interface

**Package**: `com.bienes.interfaces`

```java GuiaDAO.java
public interface GuiaDAO {
    public List<GuiayOrdendeCompra> listAll();
    public int save(GuiayOrdendeCompra bean);
    public int update(GuiayOrdendeCompra bean);
    public int delete(int cod);
    public GuiayOrdendeCompra findById(int cod);
}
```

### Implementation

**Package**: `com.bienes.dao`  
**Class**: `MySqlGuiaDAO`  
**Location**: `com/bienes/dao/MySqlGuiaDAO.java`

## Database Schema

### Table: `grr-ordendecompra`

| Column | Type | Description |
|--------|------|-------------|
| `codigo` | INT (PK, AUTO_INCREMENT) | Unique order identifier |
| `proveedor` | VARCHAR | Supplier name |
| `fecha_compra` | DATE | Purchase date |
| `descripcion` | VARCHAR | Item description |
| `cantidad` | VARCHAR | Quantity |
| `precio` | DOUBLE | Price |

<Note>
The table name uses backticks in SQL queries due to the special character (hyphen) in the name: `` `grr-ordendecompra` ``
</Note>

## Usage Examples

### Creating a Purchase Order

```java
// Obtain DAO instance
DAOFactory factory = DAOFactory.getDAOFactory(DAOFactory.MYSQL);
GuiaDAO dao = factory.getGuiaDAO();

// Create order object
GuiayOrdendeCompra orden = new GuiayOrdendeCompra();
orden.setNom_provee("TechSupply S.A.");
orden.setFecha_compra(new Date(System.currentTimeMillis()));
orden.setDescripcionguiacompra("Laptops Dell Latitude 5420");
orden.setCantidadguiacompra("10 unidades");
orden.setPrecioguiacompra(25000.00);

// Save to database
int result = dao.save(orden);
if(result > 0) {
    System.out.println("Purchase order created successfully");
}
```

### Retrieving All Orders

```java
GuiaDAO dao = factory.getGuiaDAO();
List<GuiayOrdendeCompra> ordenes = dao.listAll();

for(GuiayOrdendeCompra orden : ordenes) {
    System.out.println("Order #" + orden.getCodigoguiacompra());
    System.out.println("Supplier: " + orden.getNom_provee());
    System.out.println("Total: $" + orden.getPrecioguiacompra());
    System.out.println("---");
}
```

### Updating Order Price

```java
GuiaDAO dao = factory.getGuiaDAO();
GuiayOrdendeCompra orden = dao.findById(123);

if(orden != null) {
    orden.setPrecioguiacompra(orden.getPrecioguiacompra() * 1.10); // 10% increase
    int result = dao.update(orden);
    if(result > 0) {
        System.out.println("Order price updated");
    }
}
```

## Data Considerations

### Quantity Field

<Note>
The `cantidad` field is stored as `VARCHAR` instead of numeric type, allowing for values like "10 unidades" or "5 cajas". Validate and parse accordingly.
</Note>

### Table Naming

The table name `` `grr-ordendecompra` `` requires backticks in SQL queries due to the hyphen character. This is handled automatically in all DAO methods.
