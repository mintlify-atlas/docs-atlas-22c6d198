## Overview

The `BienesDAO` interface and its MySQL implementation provide CRUD operations for managing **bienes** (assets/goods) in the inventory system. Each asset is linked to a supplier and purchase order, enabling complete traceability.

### Interface

**Package**: `com.bienes.interfaces`

```java BienesDAO.java
public interface BienesDAO {
    public int save(Bienes bean);
    public int update(Bienes bean);
    public int delete(int cod);
    public Bienes findById(int cod);
    public List<Bienes> listAll();
    public List<Bienes> listarBienesporCodigodeComptra(int cod);
}
```

### Implementation

**Package**: `com.bienes.dao`  
**Class**: `MySqlBienesDAO`  
**Location**: `com/bienes/dao/MySqlBienesDAO.java`

## Database Schema

### Table: `bienes`

| Column | Type | Description |
|--------|------|-------------|
| `codigo_bien` | INT (PK, AUTO_INCREMENT) | Unique asset identifier |
| `descrip_bien` | VARCHAR | Asset description |
| `cantidad_bien` | INT | Quantity |
| `nom_provee` | VARCHAR | Supplier name |
| `fecha_llegada` | DATE | Arrival date |
| `codigo_ordencompra` | INT (FK) | Purchase order reference |

### listarBienesporCodigodeComptra()

Retrieves assets filtered by purchase order code (custom query with JOIN).

```java
public List<Bienes> listarBienesporCodigodeComptra(int cod)
```

<ParamField path="cod" type="int" required>
  Purchase order code (codigo) from `grr-ordendecompra` table
</ParamField>

<ResponseField name="return" type="List<Bienes>">
  List of assets associated with the specified purchase order
</ResponseField>

#### SQL Query

```sql
SELECT b.codigo_bien, 
       b.descrip_bien, 
       b.cantidad_bien, 
       b.nom_provee, 
       b.fecha_llegada, 
       b.codigo_ordencompra 
FROM bienes b 
JOIN `grr-ordendecompra` a ON b.codigo_ordencompra = a.codigo 
WHERE a.codigo=?
```

#### Implementation

```java MySqlBienesDAO.java:211-249
@Override
public List<Bienes> listarBienesporCodigodeComptra(int cod) {
    List<Bienes> lista = new ArrayList<Bienes>();
    Connection cn = null;
    PreparedStatement pstm = null;
    ResultSet rs = null;
    try {
        cn = MySqlConexion.getConectar();
        String sql = "select b.codigo_bien, b.descrip_bien, b.cantidad_bien, b.nom_provee, " +
                     "b.fecha_llegada, b.codigo_ordencompra from bienes b " +
                     "join `grr-ordendecompra` a on b.codigo_ordencompra = a.codigo " +
                     "where a.codigo=?";
        pstm = cn.prepareStatement(sql);
        pstm.setInt(1, cod);
        rs = pstm.executeQuery();
        while(rs.next()) {
            Bienes Bien = new Bienes();
            Bien.setCodigo_bien(rs.getInt(1));
            Bien.setDescrip_bien(rs.getString(2));
            Bien.setCantidad_bien(rs.getInt(3));
            Bien.setNom_provee(rs.getString(4));
            Bien.setFecha_llegada(rs.getDate(5));
            Bien.setCodigoOrdendeCompra(rs.getInt(6));
            lista.add(Bien);
        }
    } catch (SQLException e) {
        e.printStackTrace();
    } finally {
        try {
            if(rs != null) rs.close();
            if(pstm != null) pstm.close();
            if(cn != null) cn.close();
        } catch (SQLException e2) {
            e2.printStackTrace();
        }
    }
    return lista;
}
```

## Usage Examples

### Creating a New Asset

```java
// Obtain DAO instance
DAOFactory factory = DAOFactory.getDAOFactory(DAOFactory.MYSQL);
BienesDAO dao = factory.getBienesDAO();

// Create asset object
Bienes bien = new Bienes();
bien.setDescrip_bien("Laptop Dell Latitude 5420");
bien.setCantidad_bien(10);
bien.setNom_provee("TechSupply S.A.");
bien.setFecha_llegada(new Date(System.currentTimeMillis()));
bien.setCodigoOrdendeCompra(123);

// Save to database
int result = dao.save(bien);
if(result > 0) {
    System.out.println("Asset saved successfully");
}
```

### Retrieving Assets by Purchase Order

```java
BienesDAO dao = factory.getBienesDAO();
List<Bienes> bienes = dao.listarBienesporCodigodeComptra(123);

for(Bienes bien : bienes) {
    System.out.println(bien.getDescrip_bien() + " - " + bien.getCantidad_bien());
}
```

### Updating Asset Information

```java
BienesDAO dao = factory.getBienesDAO();
Bienes bien = dao.findById(45);

if(bien != null) {
    bien.setCantidad_bien(bien.getCantidad_bien() + 5);
    int result = dao.update(bien);
    if(result > 0) {
        System.out.println("Asset updated successfully");
    }
}
```
