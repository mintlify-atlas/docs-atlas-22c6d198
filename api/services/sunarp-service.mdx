## Overview

**Package:** `com.bienes.services`

**Full Class Name:** `com.bienes.services.InscripcionSUNARPServices`

The `InscripcionSUNARPServices` class provides business logic for managing SUNARP (Superintendencia Nacional de los Registros Públicos) inscriptions in Peru. This service handles the registration and management of goods/assets that need to be registered with the national public registry.

### Dependencies

- **DAO Factory:** `DAOFactory.getDAOFactory(1)` - MySQL implementation
- **DAO Interface:** `InscripcionSUNARPDAO`
- **Entity:** `com.bienes.entidad.InscripcionSUNARP`

### Registering a New SUNARP Inscription

```java
InscripcionSUNARP bean = new InscripcionSUNARP();
bean.setDescrip_bien("Vehiculo Toyota Camry 2023");
bean.setCantidad_bien(1);
bean.setPrecio_bien(85000.00);

    System.out.println("SUNARP inscription registered successfully");

    System.out.println("Failed to register SUNARP inscription");
}
```

## Error Handling

The service layer returns integer values for CUD operations:

- **Success:** Return value `> 0` indicates the number of affected rows
- **Failure:** Return value `<= 0` indicates operation failed

### Common Error Scenarios

1. **Invalid Price:** `precio_bien` must be a valid positive double value
   ```java
   if (precio <= 0) {
       throw new IllegalArgumentException("Price must be positive");
   }
   ```

2. **Missing Provider:** `nom_provee` should reference a valid provider
   ```java
   Proveedor provider = proveedorService.buscarPorNombre(nom_provee);
   if (provider == null) {
       // Handle invalid provider
   }
   ```

3. **Invalid Date:** `fecha_llegada` must be a valid SQL Date
   ```java
   try {
       Date fecha = Date.valueOf(fechaStr);
   } catch (IllegalArgumentException e) {
       // Handle invalid date format
   }
   ```

4. **Entity Not Found:** When updating or deleting a non-existent inscription
   ```java
   InscripcionSUNARP bean = servicio.buscarPorId(cod);
   if (bean == null) {
       System.out.println("Inscription not found");
       return;
   }
   ```

### Database Constraints

- Primary key constraint on `codigo_inscri`
- Possible foreign key constraint on provider name
- Not null constraints on required fields

## Business Context

### SUNARP Overview

SUNARP (Superintendencia Nacional de los Registros Públicos) is Peru's national public registry authority. The system handles:

- **Property Registry:** Real estate and movable assets
- **Vehicle Registry:** Automobile and vehicle ownership
- **Legal Entity Registry:** Business and organization registration

### Use Cases

1. **Asset Registration:** Register newly acquired assets that require public registry
2. **Price Tracking:** Maintain valuation history of registered assets
3. **Provider Documentation:** Link assets to their providers for audit trails
4. **Compliance:** Ensure all significant assets are properly registered

### Data Integrity

```java
// Validate before registration
public boolean validarInscripcion(InscripcionSUNARP bean) {
    if (bean.getPrecio_bien() <= 0) return false;
    if (bean.getCantidad_bien() <= 0) return false;
    if (bean.getDescrip_bien() == null || bean.getDescrip_bien().trim().isEmpty()) return false;
    if (bean.getNom_provee() == null) return false;
    return true;
}
```
